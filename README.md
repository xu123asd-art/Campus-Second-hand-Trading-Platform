\# 校园二手交易平台 (Campus-Market)



\## 项目概述

本项目旨在开发一款适配校园场景的二手物品交易系统，解决校园闲置资源流转不畅、信息不对称等问题，目标用户为全体在校师生及校园管理工作人员。



\## 核心功能介绍

1\. 用户管理：实现用户注册、账号登录、实名/学号认证、密码修改及信用评价体系。

2\. 商品模块：支持图文并茂的闲置物品发布、商品分类浏览、关键词全局检索。

3\. 交易与互动：提供商品留言板、买卖双方私信沟通功能。

4\. 订单追踪：完整的订单状态流转（拍下、付款、面交/发货、确认收货）。

5\. 后台管理：管理员可进行违规商品审核与下架、用户账号异常封禁与管理。
